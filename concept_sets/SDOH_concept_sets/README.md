
# Social Determinants of Health (SDOH) Concept Sets

## Overview

We reviewed two major SDOH screening initiatives and semantically mapped their questions and response options to OMOP standard concepts.

The resulting concept sets are designed to be reusable and interoperable within the OMOP Common Data Model framework.

## Concept Set Categories

The concept sets are organized under the following categories:

- **Transportation Barrier Concepts**
- **Housing Instability Concepts**
- **Food Insecurity Concepts**

## Intended Use

These concept sets support standardized representation of SDOH screening data and enable consistent analytics across sites using OMOP.
